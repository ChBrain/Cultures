# Perplexity Space

Run a culture in a Perplexity Space.

## Setup

1. Download the targeted zip for this deployment (see below).
2. Create a Space and add the extracted culture files.
3. Apply `instructions.md` from this folder.
4. Include `stack.md` and the engine scaffold files in the Space.

The Perplexity instruction set is being finalized -- `instructions.md` is a
placeholder until then.
