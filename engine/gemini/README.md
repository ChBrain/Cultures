# Gemini Gem

Run a culture as a Gemini Gem.

## Setup

1. Download the targeted zip for this deployment (see below).
2. Create a Gem and add the extracted culture files as knowledge.
3. Paste `instructions.md` from this folder into the Gem instructions
   as-is; do not edit or summarize it.
4. Include `stack.md` and the engine scaffold files as knowledge.

The instructions define how the personas operate within the cultural
architecture. They must be exact.
