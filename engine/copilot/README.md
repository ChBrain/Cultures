# Copilot Project

Run a culture as a GitHub Copilot project.

## Setup

1. Download the targeted zip for this deployment (see below).
2. Add the extracted culture files to your Copilot project.
3. Paste `instructions.md` from this folder in as-is; do not edit or
   summarize it.
4. Include `stack.md` and the engine scaffold files alongside the culture
   files.

The instructions define how the personas operate within the cultural
architecture. They must be exact.
