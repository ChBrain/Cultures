Run a culture in Google NotebookLM.

## Setup

1. Download the targeted zip for this deployment (see below).
2. Create a notebook and add the extracted culture files as sources.
3. Apply `instructions.md` from this folder.
4. Include `stack.md` and the engine scaffold files as sources.

The NotebookLM instruction set is being finalized -- `instructions.md` is a
placeholder until then.
