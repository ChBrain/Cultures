## Deployment

Each country produces a zip bundle: `khai_cultures_{country}.zip`

The Europe manifest will expand as more countries reach completion. Each time a country is added:
1. Content is audited (L0-L4g validation)
2. Country README is added
3. Europe manifest is updated
4. Regional zip grows
